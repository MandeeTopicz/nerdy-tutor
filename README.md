# NerdyTutor — Live AI Video Tutor

Real-time Socratic AI tutor with live avatar, built for the Gauntlet AI G4 program.

## Architecture
```
Browser → LiveKit Room → Agent (STT → LLM → TTS) → Simli Avatar → Browser
```

| Stage | Service |
|---|---|
| STT | Deepgram Nova-3 |
| LLM | Groq Llama-3.3-70b-versatile |
| TTS | Cartesia Sonic-3 |
| Avatar | Simli (LiveKit plugin) |
| Orchestration | LiveKit Agents 1.4.4 |
| Frontend | Next.js 16 + TypeScript |
| Session Summary | Anthropic Claude (Haiku) |

## Features

- Socratic method — never gives direct answers, always responds with a question
- Grade-adaptive prompting (6th, 8th, 10th grade)
- Live avatar with lip sync via Simli
- Live transcript panel
- Inactivity detection — check-in at 30s, graceful close at 60s
- Session summary generated by Claude after each session
- Concept tracker and latency dashboard

## Latency Benchmarks

| Stage | Measured |
|---|---|
| STT | ~529ms |
| LLM TTFT | ~280ms |
| TTS TTFB | ~200ms |
| E2E | ~1794ms |

E2E includes ~600ms Simli avatar rendering overhead on top of the STT+LLM+TTS pipeline.

## Environment Variables

Create `.env.local` in the project root:
```
DEEPGRAM_API_KEY=
GROQ_API_KEY=
CARTESIA_API_KEY=
SIMLI_API_KEY=
SIMLI_FACE_ID=
LIVEKIT_URL=
LIVEKIT_API_KEY=
LIVEKIT_API_SECRET=
ANTHROPIC_API_KEY=
```

## Running Locally

Terminal 1 — Frontend:
```bash
npm install
npm run dev
```

Terminal 2 — Agent:
```bash
cd agent
pip install -r requirements.txt
python agent.py dev
```

## Running Tests
```bash
cd agent
pytest tests/test_agent.py -v
```

## Deployment

Frontend deploys to Vercel. Add all environment variables to your Vercel project settings. The Python agent runs separately and connects to your LiveKit Cloud project.
